## About The Project

**Sparkle Clean** is a fully responsive, clean-designed landing page built for modern cleaning businesses. It focuses on user experience, high conversion rates, and features an **interactive custom JavaScript cost calculator** to estimate flat-rate cleaning prices instantly

### Key Features:
* **Interactive Pricing Calculator:** Dynamic cost estimation based on service type and number of bedrooms using optimized vanilla JavaScript
* **Mobile-First Responsive Design:** Built using **Bootstrap 5**, ensuring an elite look on iPhones, iPads, and desktop screens
* **Conversion-Oriented Elements:** Implements highly effective US marketing call-to-actions (CTAs) like *"Get a Free Quote"*
* **Performance & Clean Code:** Built following modern web development standards (including strict mode JS) with zero heavy third-party dependencies

---

## Tech Stack

* **Frontend:** HTML5, CSS3, Bootstrap 5
* **Logic:** Vanilla JavaScript (`'use strict'`)
